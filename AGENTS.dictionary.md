# AGENTS.dictionary.md

The shared dictionary between the AI and the dev. When a word here applies, use it — don't invent synonyms.

## Requirements

- Keep definitions short: one or two sentences.
- One entry per word, alphabetically ordered.
- Name what to avoid when the wrong synonym is tempting.

## How to add a word

1. Add an entry under **Words** in alphabetical order, using the format below.
2. If it replaces or conflicts with an existing word, update or remove the old entry — never keep two words for the same thing.

Format:

```markdown
**Word**:
Short definition.
_Avoid_: synonyms not to use
```
